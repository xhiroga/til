### URL

> [!NOTE] RFC2616はURLの最大長を定めていないけど、長すぎる場合にDNS問い合わせやHTTPリクエストで問題にならないの？

#### URLフラグメント

URLにおいて`#`から始まる文字列。IDを加えたURLにリンクすることで特定の説にリンクできる。これを文書フラグメントと呼ぶ。

また、Webページ内のIDが指定されていない節でも、テキストを指定してリンクすることができ、これをテキストフラグメントと呼ぶ。
