ボックスレイアウトは横書き文章の要領のレイアウト。肝は3つ。
1. 新しい行から始めるか
2. 挿絵(浮動要素)にするかどうか
3. 挿絵に隣接させたいかどうか

HTMLの全ての要素は、前後に新しい行を伴う"ブロックレベル要素"と同じ行の中で始まる"インライン要素"に分けられる。
