floatsを無視してコンテンツを中央揃えすることはできない
単純にtext-align:center;を指定しても、コンテンツが親ボックスの中央に配置されることはない。float:right;は、親ボックスの内側のコンテンツ表示領域を侵食するからである。
中央に配置されるには、以下の3ステップを踏む必要がある。
1. 親要素をposition:relative;に、子要素をposition:absolute;にする。
2. 子要素にtext-align:center;とwidth:XX%を指定し、かつ親要素のマージンをマイナス方向のマージンで相殺する。
3. 子要素の内側にテキストを配置する。
