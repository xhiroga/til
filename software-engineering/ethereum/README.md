# Ethereum

## ネットワーク
Genesisブロック作成時＋クライアント立ち上げ時に接続先のネットワークを指定する。  
さらに、ネットワーク探索の有効/無効をオプションで指定可能(=パブリック/プライベート)  

## Solidity

Ethereum上で動くアプリケーションのブロックを作成するContractを作成するためのコード。  
EVM Code(Ethereum上で動くバイトコード)にコンパイルされたのち実行される。
