# MCP Server with Python SDK

## Debug

```sh
mcp dev server.py
```

## Installation

## Claude Desktop

WSLではなくClaudeを実行しているWindows上で実行する必要がある。

```sh
uv run mcp install server.py --name "Demo by Python SDK"
```
