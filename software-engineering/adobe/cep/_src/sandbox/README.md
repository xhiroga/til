## How to develop

### ZXP/UXP Installerを用いる場合

```shell
yarn install -D
yarn zxp
```

- ZXPファイルをZXP/UXP InstallerにDrag
  - ファイルをウィンドウ上に移動してもDrag&Dropが有効にならない場合、Installer の再起動で直った
- インストール後、After Effects の Window > Extensions がグレーアウトしたままの場合、After Effect の再起動で直った
