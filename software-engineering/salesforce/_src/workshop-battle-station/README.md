# Salesforce REST APIを実行する

## Concept

### オブジェクト

- RDBのテーブル、MongoDBのコレクションに相当する概念と思われる
- FilemakerやAirTable同様、組み込みの便利機能をサポートしている。
  - Page Layouts
  - Triggers
  - Validation Rules

### アプリケーション

- オブジェクトを操作するアプリケーション
- 自分がどのアプリケーションを開いているかは、URLの `https://{HOST}/ligntning/{APPLICATION}` で判別できる
