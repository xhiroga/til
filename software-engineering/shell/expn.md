# 変数展開

シェルスクリプトで出てくる記号について

## 変数展開(parameter expansion)

```shell
: 未定義かどうか、でなく、値が空かどうか、で判断する。
- 未定義時に代用
= 未定義時に代入
+ 定義時に代用
```
