# Rust

プログラミング言語。Firefox QuantumのCSSエンジンなどに使われている。


```shell
rustc main.rs
./main
  
# RustにおけるmavenがCargo
cargo build
cargo run
```

プロジェクト用ディレクトリ直下の.toml ファイルを参照する.。常時はdebugビルドなので、--releaseをつけると最適なビルドをする。
