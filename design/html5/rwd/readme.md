# レスポンシブWebデザイン

あらゆるプラットフォームに対応する:
* ウィンドウサイズに応じて画面, イメージが可変...Fluid Layout
* ディスプレイ特性に応じて別のCSS...Media Query

# CSSをマークアップするためのコツ

* font-sizeはemで指定する
親要素のフォントサイズに対する倍率として指定する
